Mock Server

为前后端协作打造的轻量级本地 Mock 服务 · VS Code 插件

安装 / 启动 功能

在 VS Code 中快速编写、管理与运行本地 Mock API

基于项目根目录的 .mock/ YAML 配置，自动注册路由并支持多响应 rule、热更新与可视化编辑。

立即开始 查看文档
YAML 示例
name: Get User Info
description: 获取用户基本信息
method: GET
endpoint: /api/user
rules:
  - name: 正常返回
    status: 200
    headers:
      Content-Type: application/json
    body:
      id: 1
      name: Leo
    delay: 0

核心功能

自动路由注册

扫描 .mock/ 下的 YAML 文件，自动生成本地路由，立即可用。

多响应规则

为同一路由定义多个 rule（正常、出错、延迟等），在 UI 中一键切换。

WebView 可视化编辑

在 VS Code 的侧边栏打开编辑面板，所见即所得地维护 YAML 配置。

热更新与文件监听

保存 YAML 即可触发 Mock Server 重载，快速迭代体验。

安装与使用

  1. 在 VS Code 中打开项目根目录。
  2. 按 Cmd+P 输入 Extensions: Install Extension，搜索并安装本插件（或从源码加载）。
  3. 在侧边栏打开 Mock Server 面板，点击 Start 启动（默认端口 9527）。
  4. 将你的 YAML 文件放入项目根目录的 .mock/，并在浏览器或前端中使用 http://localhost:9527 发起请求。

提示：可以在插件设置中修改端口、自动启动等参数。
